---
title: Deployment setup
---

Smelter can be deployed in various ways, depending on your platform, used features and whether it is used standalone or via Membrane Framework plugin.

Consider the following options for Smelter deployment:
### Docker
    - (recommended) <a href="https://github.com/software-mansion/smelter/blob/master/build_tools/docker/slim.Dockerfile" target="_blank">Dockerfile</a> with compositor without web rendering support
    - <a href="https://github.com/software-mansion/smelter/blob/master/build_tools/docker/full.Dockerfile" target="_blank">Dockerfile</a> with compositor with web rendering support
### Standalone binaries
    - Building <a href="https://github.com/software-mansion/smelter" target="_blank">from source</a>.
    - Using binaries from <a href="https://github.com/software-mansion/smelter/releases" target="_blank">GitHub releases</a>.
### Element in a Membrane pipeline. 
<a href="https://compositor.live/docs/deployment/overview#membrane-framework-plugin" target="_blank">Learn more</a>.
