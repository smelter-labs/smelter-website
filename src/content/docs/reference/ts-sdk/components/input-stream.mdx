## Props

### inputId
ID of an input. It identifies a stream registered using a [LiveCompositor.registerInput]() method.

- Type: `string`

---

### id
ID of a component.

- Type: `string`
- Default value: Value produced by `useId` hook

---

### volume
Audio volume of the input stream. Accepts values between 0 and 1.

- Type: `number`
- Default value: 1

---

### mute
Determines whether to mute audio of the input stream.

- Type: `boolean`
- Default value: `false`
