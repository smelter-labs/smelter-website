`useAudioInput` can be used to control audio configuration. If you're using the `InputStream` component, you can use the `mute` and `volume` properties to control the audio instead.

<Aside type="caution">
  Adding this hook or `<InputStream />` component more than once or for the
  specific input will sum the configured volume.
</Aside>

## Reference

```tsx
function useAudioInput(inputId: string, audioOptions: AudioOptions);
```

## Arguments

### inputId

ID of an input. It identifies a stream registered using a [Smelter.registerInput](/ts-sdk/1.0/smelter#registerinput) method.

* **Type**: `string`

***

### audioOptions

`volume` - Number between 0 and 1 representing the audio volume.

* **Type**: `{ volume: number }`
