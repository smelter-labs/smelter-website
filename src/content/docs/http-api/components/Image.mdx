A component for rendering images:
- From external URLs
- From the server hosting a Smelter instance

:::note
To use this component, you need to first register the image with matching `image_id` using [`register image`](/http-api/routes#register-image) request.
:::

## Reference


### Example

<CollapsibleDetails summaryTitle='Register png image' open>
<Code code={`
POST: /api/image/example_png/register
Content-Type: application/json

{
    "asset_type": "png",
    "url": "https://example.com/image.png"
}
`} lang='http' />
</CollapsibleDetails>


<CollapsibleDetails summaryTitle='Register RTP output with registered image "example_png"' open>
<Code ins={[16, 17]} code={`
POST: /api/output/example_output/register
Content-Type: application/json

{
    "type": "rtp_stream",
    "port": 8002,
    "ip": "127.0.0.1",
    "video": {
        "resolution": { "width": 1920, "height": 1080 },
        "encoder": { "type": "ffmpeg_h264", "preset": "ultrafast" },
        "initial": {
            "root": {
                "type": "rescaler",
                "mode": "fit",
                "child": {
                    "type": "image",
                    "image_id": "example_png"
                }
            }
        }
    }
}
`} lang='http' />
</CollapsibleDetails>

Now you can see your image streamed on port 8004.

## Properties

### id
ID of a component.

- **Type**: `string`

---

### image_id
ID of an image. It identifies an image registered using a [register image](/http-api/routes#register-image) request.

- **Type**: `string`
