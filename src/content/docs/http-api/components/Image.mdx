A component for rendering images:
- From external URLs
- From the server hosting a Smelter instance

<Aside> To use this component, you need to first register the image with matching `image_id` using [`register image`](/http-api/routes#register-image) request. </Aside>

## Example

Before using an Image in a stream, it must first be registered via an HTTP request. Use the following request to register a PNG image:

<CollapsibleDetails summaryTitle='Register PNG image' open>
```http
POST: /api/image/example_image/register
Content-Type: application/json

{
  "asset_type": "svg",
  "url": "https://example.com/image.svg"
}
```
</CollapsibleDetails>

Once this request is successfully processed, the Image will be available for use in the output stream. You can reference a specific Image by its unique `image_id` (in this example `example_image`)

<CollapsibleDetails summaryTitle='Update output' open>
```http
POST: /api/output/example_output/update
Content-Type: application/json

{
  "video": {
    "root": {
      "type": "view",
      "background_color": "#52505b",
      "children": [
        {
          "type": "image",
          "image_id": "example_image"
        }
      ]
    }
  }
}
```
</CollapsibleDetails>

<CollapsibleDetails summaryTitle='Output'>
  <img src={ExampleImage.src} />
</CollapsibleDetails>
