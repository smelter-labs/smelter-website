`WebView` renders a website using Chromium engine embedded inside the Smelter.

<Aside>To use this component, you need to first register the image with a matching `instance_id` using the [`register web renderer`](/http-api/routes/#register-web-renderer-instance) function.</Aside>

<Aside type='caution'>Only one component can use specific `instance_id` at the time.</Aside>

## Properties

### id
ID of component.

- **Type**: `string`

### children
List of component's children.

- **Type**: `Component[]`

### instance_id
ID of a web renderer instance. It identifies an instance registered using a
  [`register web renderer`](/http-api/routes#register-web-renderer-instance) request.

- **Type**: `string`

<Aside type='caution'>You can only refer to specific instances in one Component at a time.</Aside>
