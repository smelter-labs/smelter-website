Represents a font file uploaded through a `multipart/form-data` request to the Smelter. Used to dynamically load and register custom fonts.

<CollapsibleDetails summaryTitle='Type definitions' open>
<Code code={`
type Font = Multipart
`} lang='tsx' />
</CollapsibleDetails>

The field name in a Multipart can be arbitrary, but it must contain a binary font file.
