## Reference
<CollapsibleDetails summaryTitle='Type definitions' open>

```tsx
type VideoEncoderOptions = {
  bitrate?: VideoEncoderBitrate;
  keyframe_interval_ms?: number;
  preset?: VideoEncoderPreset[];
  pixel_format?: "yuv420p" | "yuv422p" | "yuv444p";
  ffmpeg_options?: Map<string, string>;
}
```
</CollapsibleDetails>

### Properties

#### bitrate
Bitrate of the output stream, in bits per second [b/s].
If unset the default value of the encoder is used.

- **Type**: [`VideoEncoderBitrate`](#videoencoderbitrate)
- **Supported encoders**: `ffmpeg_h264`, `ffmpeg_vp8`, `ffmpeg_vp9`, `vulkan_h264`

---

#### keyframe_interval_ms
Interval between keyframes, in milliseconds.

- **Type**: `number`
- **Default value**: `5000`
- **Supported encoders**: `ffmpeg_h264`, `ffmpeg_vp8`, `ffmpeg_vp9`

---

#### preset
Defines encoder tuning.

- **Type**: [`VideoEncoderPreset`](#videoencoderpreset)
- **Default value**: `low_latency_balanced`
- **Supported encoders**: `ffmpeg_h264`, `ffmpeg_vp8`, `ffmpeg_vp9`

---

#### pixel_format
Encoder pixel format

- **Type**: `"yuv420p" | "yuv422p" | "yuv444p"`
- **Default value**: `yuv420p`
- **Supported values**:
  - `yuv420p`
  - `yuv422p`
  - `yuv444p`
- **Supported encoders**: `ffmpeg_h264`, `ffmpeg_vp9`

---

#### ffmpeg_options
Raw FFmpeg encoder options. Visit <a href='https://ffmpeg.org/ffmpeg-codecs.html' target='_blank'>FFmpeg docs</a> to learn more.

- **Type**: `Map<string, string>`
- **Supported encoders**: `ffmpeg_h264`, `ffmpeg_vp8`, `ffmpeg_vp9`

---

## VideoEncoderBitrate

<CollapsibleDetails summaryTitle='Type definitions' open>

```tsx
type VideoEncoderBitrate = { average_bitrate: number; max_bitrate: number; } | number;
```
</CollapsibleDetails>

Bitrate in bits/second. If only the `number` is specified it is treated as average bitrate
and max bitrate is set to 1.25 of the provided value.

### Properties

#### average_bitrate
Average bitrate in bits/second. Encoder will try to keep the bitrate around the provided average, but may temporarily increase it to the provided max bitrate.

- **Type**: `number`

---

#### max_bitrate
Max bitrate in bits/second.

- **Type**: `number`

---

## VideoEncoderPreset

<CollapsibleDetails summaryTitle='Type definitions' open>

```tsx
type VideoEncoderPreset = "quality" | "balanced" | "fast" | "low_latency"
```
</CollapsibleDetails>
Encoder tuning presets.

### Properties

- **Supported values**:
  - `quality` - prioritizing encoding quality at the expense of speed
  - `balanced` - compromise between speed and quality
  - `fast` - prioritizing fast encoding at the expense of quality
  - `low_latency` - tune encoder for low latency streaming
