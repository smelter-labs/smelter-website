A component for displaying registered media inputs.

<Aside>To use this component, you need to first register the image with a matching `inputId` using the [`Smelter.registerInput`](/ts-sdk/smelter#registerinput) function.</Aside>

## Props

### inputId
ID of an input. It identifies a stream registered using a [Smelter.registerInput](/ts-sdk/smelter#registerinput) method.

- **Type**: `string`

---

### id
ID of a component.

- **Type**: `string`
- **Default value**: Value produced by `useId` hook

---

### volume
Audio volume of the input stream. Accepts values between 0 and 1.

- **Type**: `number`
- **Default value**: 1

---

### mute
Determines whether to mute audio of the input stream.

- **Type**: `boolean`
- **Default value**: `false`
