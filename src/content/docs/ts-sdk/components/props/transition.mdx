```tsx
type Transition {
  durationMs: number;
  easingFunction?: EasingFunction | null;
  shouldInterrupt?: boolean;
}
```
</CollapsibleDetails>

## Properties

### durationMs
Duration of a transition in milliseconds.

- **Type**: `number`

---

### easingFunction
Easing functions are used to interpolate between two values over time during transition.

- **Type**: [`EasingFunction`](/ts-sdk/components/props/easing-function)
- **Default value**: `"linear"`

---

### shouldInterrupt
If `true`, instead of continuing currently ongoing transition, new transition will start from the current state

- **Type**: `boolean`
- **Default value**: `false`
