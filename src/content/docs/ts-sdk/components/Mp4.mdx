A component that renders content of Mp4 file. It is simpler alternative to the [`InputStream`](/ts-sdk/components/inputstream)
component that does not require registration, but with more limited set of options.

<Aside type='caution'>`@swmansion/smelter-web-wasm` currently does not support audio from mp4 files.</Aside>

## Props

### source 
Url or local path to the mp4 file. Note that the path has to be local to the machine where
Smelter server runs.

- **Type**: `string`

---

### volume
Audio volume of the input stream. Accepts values between 0 and 1.

- **Type**: `number`
- **Default value**: 1

---

### muted
Determines whether to mute audio of the input stream.

- **Type**: `boolean`
- **Default value**: `false`
