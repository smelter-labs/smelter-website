A component for rendering images:
- From external URLs
- From the server hosting a Smelter instance

<Aside>To use this component, you need to first register the image with a matching `imageId` using the [`Smelter.registerImage`](/ts-sdk/smelter#registerimage) function.</Aside>

## Props

### imageId
ID of an image. It identifies an image registered using a [Smelter.registerImage](/ts-sdk/smelter#registerimage) method.

- **Type**: `string`

---

### id
ID of a component.

- **Type**: `string`
- **Default value**: Value produced by `useId` hook
