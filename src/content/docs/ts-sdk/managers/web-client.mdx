Connects to remotely running Smelter instance from a browser.

<Code title='RemoteSmelterExample.tsx' code={`
import Smelter from "@swmansion/smelter-web-client"

async function run() {
    const smelter = new Smelter({ url: 'http://192.168.1.33:8081' });
    await smelter.init();
}
void run();
`} lang="tsx" />

### Properties

#### url

Address of a running smelter server. Protocol used in the address also implies protocol of a WebSocket connection (`http -> ws` and `https -> wss`).

- **Type**: `string | URL`
